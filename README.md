# Enterprise Asset Lifecycle Management System

## MSc IT Project

**Student:** Xiaoyun Chen

**University:** University of Glasgow

### Project Overview

This project aims to develop a web-based Enterprise Asset Lifecycle Management System for managing enterprise assets throughout their complete lifecycle.

The system will support:

- Asset Registration
- Asset Transfer
- Maintenance Management
- Depreciation Calculation
- Inventory Audit
- Asset Disposal
- Reporting Dashboard

Lightweight AI-assisted features will be included, such as:

- Natural Language Asset Search
- Idle Asset Analysis
- Intelligent Maintenance Reminder

The project is being developed as part of the MSc IT Project at the University of Glasgow.
